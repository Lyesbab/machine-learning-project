# Obesity Level Prediction using Machine Learning

## Overview
This academic project applies supervised machine learning techniques to predict obesity levels
based on eating habits, lifestyle, and physical characteristics.

## Dataset
- Source: UCI Machine Learning Repository  
- Dataset: Estimation of Obesity Levels Based on Eating Habits and Physical Condition  
- Target variable: NObeyesdad (7 obesity classes)

## Methodology
- Exploratory Data Analysis (EDA)
- Data preprocessing (StandardScaler, OneHotEncoder)
- Train / Validation / Test split (60% / 20% / 20%)
- Machine learning pipelines using scikit-learn

## Models Used
- Logistic Regression
- K-Nearest Neighbors (KNN)
- Decision Tree
- Random Forest
- Gradient Boosting
- Support Vector Machine (SVM)
- MLPClassifier (Neural Network)

## Results
- Best model: Optimized MLPClassifier
- Test accuracy: ~97%
- Strong generalization performance
